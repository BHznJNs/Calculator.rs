# Calculator.rs

An simple command-line calculator program writen with Rust.

## Features

- Variable support

```text
> a = 1
> a
= 1
```

- Math function support

```text
> sin(1) 
= 0.84147098
```

## Build-ins

- Functions
  - sin
  - cos
  - tan
  - asin
  - acos
  - atan
  - sinh
  - cosh
  - tanh

  - rad
  - deg

  - log10
  - log2
  - ln
  - exp

  - abs
